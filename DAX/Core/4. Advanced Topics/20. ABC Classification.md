## 20. **ABC Classification (Pareto)**  
    ```DAX 
    ABC Class = 
    VAR CumulativeSales = 
      SUMX(TOPN(100, VALUES(Product[ID]), [Total Sales], DESC), [Total Sales])
    RETURN
      IF(CumulativeSales <= 0.8, "A", IF(CumulativeSales <= 0.95, "B", "C")) 
    ```  
    *Applies the 80/20 rule in DAX.*
