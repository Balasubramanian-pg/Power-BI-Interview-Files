## 57. **Consecutive Growth Streaks**  
    ```DAX
    Growth Streak = 
    VAR CurrentSales = [Total Sales]
    VAR PriorSales = CALCULATE([Total Sales], PREVIOUSMONTH('Date'[Date]))
    RETURN
    IF(
      CurrentSales > PriorSales, 
      [Growth Streak] + 1, 
      0
    )
    ```
    *Uses recursion-like logic (requires a calculated column).*
